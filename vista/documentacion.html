Documentacion

Usuarios

Desde la aplicación se pueden:

  • Crear más usuarios, solo puede crearlos el perfil (administrador).
  • Modificar usuarios, solo puede modificarlos el perfil (administrador).
  • Consultar usuarios, cualquier perfil.

Pqrs

Desde la aplicación se pueden:

  • Crear pqrs (incidentes o requerimientos), cualquier perfil, (solo se pueden asignar a agentes).
  • Consultar pqrs (cualquier usuario).
  • Agregar un comentario y cambiar de estado, (administrador, agente).
  • si la pqr esta cerrada no se puede agregar comentarios ni cambiar de estado.

Campos

Los campos para escribir los id tiene la función de autocompletado, cada vez que se pulsa una tecla se realizar una búsqueda en la base de datos coincidencias con la id que se está escribiendo. Cuando se escribe una ID al cambiar de input, realiza una búsqueda en la base de datos, y llena automáticamente los campos, como (nombres, apellidos, correo, etc.)
